# crew-flags Chrome Extension

Minimal Manifest V3 extension scoped to:

- `https://crew.preply.com/*`
- `https://crew.stage*.preply.org/*` (for example `crew.stage12.preply.org`)

For now clicking extension action opens an empty popup.

## Load unpacked

1. Open `chrome://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select folder: `extensions/crew-presence`
